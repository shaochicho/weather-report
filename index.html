Printed weather report-METAR飛行氣象資訊判讀自主學習

簡介：Printed weather report-METAR 的用途

METAR是飛行員接收天氣的重要工具，由一串代碼所組成，每30分鐘會更新一次，如果在三十分鐘內有顯著的天氣變化，會發布"SPECI"這個報文。這些報告提供了關於風向、風速、能見度、雲層狀況、溫度、露點和氣壓等關鍵天氣資訊，幫助飛行員做出安全的飛行決策。

METAR資訊報告結構

METAR報告通常以標準的 METAR格式呈現。即使是語音廣播(ATIS)，其核心數據也是依據此格式編碼的。

範例報告： METAR RCSS 110600Z 09009KT 050V110 9999 FEW020 SCT035 25/16 Q1018 NOSIG RMK A3007=

報告元素逐項解讀

  1. 機場代碼與時間：
    • RCSS： 機場 ICAO 代碼。

      • 台灣主要機場 ICAO 代碼

      在閱讀氣象報告 (METAR/SPECI) 時，第一個四字代碼即代表報告發布的機場。

    • 110600Z： 報告發布時間。11 是日期，0600Z 是UTC時間 06:00。
  2. 風向與風速 (WIND)：

    09009KT 050V110： 風向從 090 度吹來，風速為 9 節 (KT)， 050V110 表示風向在 050 度到 110 度之間變化。

  3. 能見度 (VISIBILITY)：

    9999： 水平能見度為 10 公里以上(10 Kilometers or more)。

  4. 雲層狀況 (CLOUDS)：

    代碼由雲量和高度（百英尺為單位）組成。

    • FEW020： 少量 (Few) 雲層在 2,000 英尺。
    • SCT035： 零星 (Scattered) 雲層在 3,500 英尺。
    飛行員需特別注意 BKN/OVC (多雲/陰天) 的最低高度，這可能影響目視飛行規則 (VFR) 的最低要求。

  5. 溫度與露點 (TEMP/DEW POINT)：

    25/16： 氣溫 25 攝氏度，露點 16 攝氏度。溫差越小，空氣濕度越高，結霧的可能性越大。

    6. METAR/SPECI 代碼詳細對照表

    以下為航空氣象報告 (METAR/SPECI) 中各組代碼的意義。METAR 是例行報告(每30分鐘一報)，SPECI 是特殊報告(在天氣劇烈變化時會發布)。

    1. 基本資訊與時間

    代碼說明
    METAR / SPECI例行航空氣象報告（每半小時） / 特殊航空氣象報告（突發天氣變化）
    ICAO 四字代碼機場代碼，如 RCTP (桃園)
    時間 (e.g., 111853Z)日期 (11) 及時間 (18:53)，Z 代表協調世界時 (Zulu Time)

    2. 風 (Wind)

    代碼說明
    dddffKT風向 (ddd) 風速 (ff) 節 (KT)
    VRBffKT風向不定，風速 (ff) 節
    dddffGggKT有陣風，Ggg 為最大陣風速度 (Gust)
    dddVddd風向可變範圍 (例如 050V110)

    3. 能見度與跑道視程 (RVR)

    代碼說明
    9999能見度 ≥10 km
    CAVOKCeiling And Visibility OK → 能見度 ≥10 km，無顯著天氣，雲底 ≥5000 ft
    R23/120023 跑道 RVR (Runway Visual Range) = 1200 m
    R05/0800N/U/DN=無變化, U=上升中, D=下降中

    4. 天氣現象

    代碼由 強度/修飾符 + 現象代碼 組成（強度：-輕，+強，VC附近）。

    類別代碼說明
    降水RA雨 (Rain)
    DZ / SN毛毛雨 (Drizzle) / 雪 (Snow)
    GR / GS冰雹 (Hail, >5mm) / 小冰雹/霰
    IC / PL霰 (Ice crystals) / 冰珠 (Ice pellets)
    能見度降低BR / FG薄霧 (Mist, vis 1–5 km) / 霧 (Fog, vis <1 km)
    FU / HZ煙 (Smoke) / 霾 (Haze)
    VA / SA火山灰 (Volcanic ash) / 沙 (Sand)
    其他TS / SH雷暴 (Thunderstorm) / 陣性 (Shower)
    FZ / FC過冷 (Freezing, 前綴) / 漏斗雲/龍捲風

    5. 雲量與雲高

    高度單位為 nnn × 100 ft。例如 040 = 4000 呎。

    雲量代碼說明
    SKC / CLR / NSC無雲 / 自動站無雲 / 無顯著雲
    FEW少量雲 (1–2 成)
    SCT疏雲 (3–4 成)
    BKN多雲 (5–7 成)
    OVC陰天 (8 成)
    VVnnn垂直能見度 (nnn×100 ft)
    CB / TCU積雨雲 (Cumulonimbus) / 龍骨狀積雲 (Towering cumulus)

    6. 溫度與氣壓

    代碼說明
    TT/DD溫度 (TT) / 露點 (DD)，皆為攝氏
    Mnn負溫度，例如 M05 = -5°C
    Qnnnn氣壓 QNH (hPa)，例如 Q1012
    Annnn氣壓 QNH (inHg)，例如 A2992

    7. 趨勢預報與備註

    代碼說明
    NOSIG無顯著變化 (No significant change)
    BECMG / TEMPO將逐漸變化 / 暫時性變化
    RMK備註 (Remarks)
    WS Rxx跑道風切變 (Wind shear)
  6. 氣壓設定 (ALTIMETER)：

    Q1018 & A3007： 高度計設定為1018 百帕或 30.07 吋汞柱。飛行員必須在進入機場區域前設定此值。

METAR 與 ATIS 的區別

雖然 METAR提供氣象數據，但 ATIS (自動終端資訊服務) 則提供更全面的機場運行資訊，包括：跑道使用(Runway condition code)、儀表進場、特殊注意事項等，以及最新的氣象數據（通常來自 METAR）。
